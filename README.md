# Okoa Pesa 💰

**Smart investing starts here**

---

## 🌍 Overview

Okoa Pesa is a modern personal finance platform designed to help individuals take control of their money through intelligent expense tracking, budgeting, and simplified investment insights.

Built with scalability in mind, Okoa Pesa bridges the gap between basic expense trackers and complex financial tools by offering a clean, intuitive, and educational experience.

---

## 💡 Problem

Many young professionals and students struggle with:

* Poor expense tracking habits
* Lack of financial literacy
* Limited access to simplified investment insights
* Overly complex financial platforms

---

## ✅ Solution

Okoa Pesa provides:

* A simple way to track daily expenses
* Smart budgeting tools
* Beginner-friendly investment insights
* Financial education integrated directly into the product

---

## 🚀 Key Features

### 📊 Smart Dashboard

* Real-time financial overview
* Budget vs spending insights
* Intelligent suggestions based on user behavior

### 💸 Expense Management

* Add, categorize, and delete expenses
* Automatic total calculations

### 🎯 Budget Tracking

* Set monthly budgets
* Visual progress indicators

### 📈 Reports & Analytics

* Interactive charts for spending patterns

### 📚 Financial Literacy Module

* Learn-by-doing experience
* Quizzes and progress tracking

### 📉 Market Insights

* External stock data integration

### 🔐 Authentication

* Secure route protection
* Session persistence

---

## 🧱 Tech Stack

* **Frontend:** React
* **Build Tool:** Vite
* **Styling:** Tailwind CSS
* **Routing:** React Router
* **State Management:** Context API
* **Charts:** Recharts
* **Backend Services:** Firebase (Auth, Firestore, Analytics)
